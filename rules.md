# VoteSamajh (वोट समझ) - Rules

## Content Accuracy Rules
- All election facts must be sourced from ECI (eci.gov.in) official documents
- No political party opinions or bias — this is strictly educational
- Statistics must include the election year they refer to
- All constitutional articles cited must reference the correct article number
- Content must be verified against the Representation of the People Act, 1950 and 1951

## Language Rules
- No hard legal language — explain every term in simple words
- If a legal term must be used, always follow it with a plain-language explanation in brackets
- Translations must be culturally appropriate, not just literal word-for-word
- Avoid transliteration where a proper native word exists

## Design Rules
- Never use political party symbols, colors, or imagery
- Never use any politician's name, photo, or quote
- Indian flag must only be used as per Flag Code of India
- EVM images used must be from official ECI sources

## Tech Rules
- No user data is collected or stored (privacy-first)
- Language preference stored in localStorage only
- No tracking scripts, no analytics that identify users
- All external links open in new tab with rel="noopener noreferrer"
